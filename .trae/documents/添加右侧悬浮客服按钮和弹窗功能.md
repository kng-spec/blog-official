# 添加右侧悬浮客服按钮和弹窗功能

## 实现步骤

1. **添加客服图标到悬浮栏**
   - 在现有的 `.rollbar` 容器中添加一个新的客服按钮
   - 使用提供的客服图标 URL
   - 为按钮添加点击事件，用于打开聊天弹窗

2. **创建聊天弹窗结构**
   - 在页面底部添加一个弹窗容器
   - 设计弹窗的基本结构，包括标题栏、关闭按钮和内容区域
   - 内容区域使用 iframe 嵌入客服聊天页面

3. **实现弹窗样式**
   - 为弹窗添加 CSS 样式，设置初始隐藏状态
   - 设计弹窗的外观，包括边框、阴影等效果
   - 确保弹窗在页面中居中显示

4. **实现响应式设计**
   - 为不同屏幕尺寸设置弹窗的宽度和高度
   - 在移动端优化弹窗大小，确保良好的用户体验

5. **添加交互功能**
   - 实现点击客服按钮打开弹窗的功能
   - 实现点击关闭按钮关闭弹窗的功能
   - 添加遮罩层，点击遮罩层也可以关闭弹窗

6. **优化用户体验**
   - 添加平滑的弹窗显示/隐藏动画
   - 确保 iframe 正确加载客服聊天页面
   - 测试在不同设备上的显示效果

## 技术要点

- 使用 HTML 和 CSS 创建弹窗结构和样式
- 使用 JavaScript 实现交互功能
- 使用 iframe 嵌入外部客服聊天页面
- 使用媒体查询实现响应式设计
- 确保代码与现有页面结构和样式兼容

## 文件修改

- 只需要修改 `dlfooter.html` 文件，无需创建新文件
- 在适当位置添加客服按钮和弹窗代码
- 确保修改不会影响现有功能
