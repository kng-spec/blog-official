### 问题分析

汉堡菜单点击无反应的原因：

1. 文件中引用了不存在的 `js/script.js`
2. 没有任何处理汉堡菜单点击事件的 JavaScript 代码
3. 虽然 CSS 中定义了 `.nav-menu.active` 和 `.hamburger.active` 类，但没有 JavaScript 来切换这些类

### 修复方案

在 `earn online.html` 文件中添加处理汉堡菜单点击事件的 JavaScript 代码，具体包括：

1. 在页面底部的 `</body>` 标签前添加 JavaScript 代码，用于：

   * 获取汉堡菜单元素和导航菜单元素

   * 为汉堡菜单添加点击事件监听器

   * 点击时切换 `.active` 类，实现菜单的显示/隐藏

   * 同时切换汉堡图标的样式，实现动画效果

### 预期效果

* 点击汉堡按钮时，导航菜单会从左侧滑出

* 再次点击汉堡按钮时，导航菜单会滑回隐藏状态

* 汉堡按钮本身会有动画效果，显示为"X"形状

* 整个交互流畅，符合移动端用户体验

### 技术细节

* 使用原生 JavaScript 实现，无需外部库

* 代码简洁高效，仅包含必要的交互逻辑

* 遵循现有的 CSS 类命名规范

* 确保与页面其他 JavaScript 代码兼容
